# Fake-News-Detection-BERT
Fake News Detection using BERT Transformer Model

## Problem Statement
Fake news has become a major issue in digital media. The objective of this project is to build a machine learning model that can automatically classify news articles as Fake or Real.

## Approach
The approach followed in this project includes:

1. Data collection from Kaggle Fake News Dataset
2. Data preprocessing and label creation
3. Text tokenization using BERT tokenizer
4. Training a transformer model for classification
5. Evaluating the model performance
6. Performing error analysis
7. Hyperparameter tuning

## Model Used
The model used in this project is:

BERT (Bidirectional Encoder Representations from Transformers)

It is a pretrained transformer model used for Natural Language Processing tasks such as text classification.

- Accuracy
